# Animated Celebration Hub

## Project info

Animated Celebration Hub is a small Vite + React + TypeScript app that demonstrates animated celebration backgrounds, event management UI, and reusable UI components using shadcn-ui and Tailwind CSS.

**Live project (Lovable)**: https://lovable.dev/projects/27cb1cf5-8696-40fe-a51e-2cc50d2154e8

## Quick start (local development)

Requirements: Node.js and npm (install via nvm if needed).

```bash
git clone <YOUR_GIT_URL>
cd animated-celebration-hub
npm install
npm run dev
```

Open the dev server URL shown by Vite (usually http://localhost:5173).

## Editing the project

- Use the Lovable editor: changes made via Lovable are committed automatically.
- Or edit locally in your preferred IDE and push changes to the repository.
